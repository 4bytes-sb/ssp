# **App Name**: Synthwave Folio

## Core Features:

- Interactive Personal Bio: Dynamically display personal information and 'About Me' details from a structured content source, allowing for easy updates.
- Skills Matrix Showcase: Visually present a comprehensive categorized list of technical skills and tools, with an engaging interface.
- Dynamic Project Portfolio: A curated, filterable list of projects, each featuring dedicated pages for detailed descriptions, technologies used, and external links.
- GitHub Profile Overview: Integrate with the GitHub API to automatically fetch and display current GitHub statistics, trophies, and top repositories.
- Integrated Social Hub: A central section for all social media and professional network links, with clear calls to action.
- AI-Powered Project Storyteller Tool: An AI tool that can generate engaging summaries or expanded narratives for projects based on different audience contexts or specific keywords.
- Direct Inquiry Contact Form: A simple contact form that allows visitors to send direct messages or inquiries, with submission stored in a database (e.g., Supabase) or routed to an email service.

## Style Guidelines:

- Dark scheme with a modern, tech-inspired feel. Primary actions and highlights in vibrant magenta (#EB47EB). Background in black. Accent elements in luminous blue-purple (#8026D9) for contrast and hierarchy. Text should use high-contrast shades of off-white and light gray.
- Both headlines and body text will use 'Space Grotesk', a sans-serif font, for a clean, contemporary, and tech-inspired aesthetic. Code snippets, if present, will utilize 'Source Code Pro', a monospaced sans-serif.
- Utilize modern, sharp, and slightly futuristic geometric icons. Icons should be monochrome or use a gradient from the color palette to maintain a cohesive visual identity.
- A responsive, modular grid-based layout ensuring adaptability across all devices. Emphasize clean lines, generous spacing, and clear segmentation between sections to improve readability and user navigation.
- Subtle, fluid transitions and micro-interactions. Elements like cards, buttons, and section headers will feature gentle hover effects and enter animations to enhance user engagement without distraction. Typing animations are also an option for titles/taglines.
